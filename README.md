# Gen-AI-with-LangChain

This project is a workspace for learning and experimenting with LangChain and LangGraph.

It will be used to explore building AI applications, orchestration patterns, and agent-based workflows.

Currently looking at working with LCEL - LangChain LangChain Expression Language (LCEL) which is a declarative, pipe-based syntax (|) for composing, streaming, and parallelizing LangChain components into AI chains.
